# ClassiCube Marketcraft - Web Deployment

This directory contains the static files for deploying ClassiCube to Vercel or any static hosting service.

## Files

- `index.html` - Main entry point with game canvas and loading UI
- `ClassiCube.js` - JavaScript glue code (generated by Emscripten)
- `ClassiCube.wasm` - WebAssembly binary (generated by Emscripten)
- `static/default.zip` - Default texture pack

## Building

The build artifacts are generated using:

```bash
make web RELEASE=1
```

Or use the automated build script:

```bash
bash scripts/build-for-vercel.sh
```

## Deployment

### Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Deploy: `vercel --prod`

### Other Static Hosts

Simply serve the `public/` directory contents.

## Usage

Open the deployed URL in a modern web browser that supports WebAssembly.

The game starts in singleplayer mode by default. To connect to a multiplayer server, modify the `Module.arguments` in `index.html`.

## Browser Requirements

- WebAssembly support (Chrome 57+, Firefox 52+, Safari 11+, Edge 16+)
- WebGL support
- Modern JavaScript (ES6+)

## Mobile Support

The webclient is fully compatible with mobile browsers (iOS Safari, Chrome on Android) and includes touch controls.
